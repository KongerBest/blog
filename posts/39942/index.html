利用hexo搭建博客（2）

Konger · 2020-1-25 · 次阅读

前言

利用hexo搭建博客（1）

这篇博文是上一篇“利用hexo搭建博客”的续集

没看过的小伙伴们

要先看看上一篇~

正文

在上一篇

我们已经做好了搭建博客的基本步骤

现在要在桌面上（随便，只是要方便打开）创建一个文件夹

叫blog（其他的也可以）

桌面右键，打开git bash

输入

$ mkdir blog

这里的blog可以替换为自己想要的名称

PS：最好路径中不要出现中文

此时，桌面上已经创建了一个名为blog的文件夹

再次输入

$ cd blog

表示进入了blog文件夹

接着输入

$ hexo init

表示初始化你的博客

等待一段时间后

再次输入

$ hexo s

表示预览你的博客

至于在hexo中常用的命令

可以看看我这篇文章

hexo常用指令一览

不关闭git bash，后台运行

在浏览器中输入

http://localhost:4000/

可以看到，搭建成功了

默认创建了一篇博文，叫做

“hello would!”

直到目前，你的博客还在本地，别人是无法访问的

所以

最最重要的一步，部署！

打开blog文件夹

找到_config.yml文件

首先，你需要注册你的github账号

点此注册

注册完后

点击“New”创建新的仓库

仓库名称为

你的名字.github.io

意思就是，假如你的名字叫Peter

那你的仓库名就应该叫做

Peter.github.io

PS：注意，大小写敏感

创建完毕后

修改如下字段：第二个字段是你的 GitHub 博客仓库地址

deploy:
   type: git
   repo: https://github.com/你的名字/你的名字.github.io
   branch: master

修改完毕后，退出保存

再次打开git bash

输入

$ hexo clean

↑ 清除缓存

$ hexo g

↑ 生成js，css代码

$ hexo d

↑ 部署到远端

注意

如果在 hexo d执行后报错，则需要配置你的用户名和邮箱

在git bash输入以下代码

$ git config --global user.name "输入你的用户名"

$ git config --global user.email "输入你的邮箱"

PS：用户名要和你github的用户名一致

​ 邮箱随意~

这样，你的快乐博客就搭建完成啦！
