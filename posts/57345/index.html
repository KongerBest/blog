github下载慢？试下FDM吧

四月 01, 2020
  • github

前言

GitHub克隆下载速度超级慢，尝试了网上的几个方法都不行，什么更改hosts和转到码云下载都不管用。以下方法亲测有效，但有时候也会下载不了，没办法……

正文

1、下载Free Download Manager，下载地址在文末

2、在GitHub下载点击download zip

3、打开下载内容页面

4、鼠标右键https那个下载链接把链接复制下来

5、打开free download manager点击主菜单，点击从粘贴板粘贴链接，然后就可以下载了，快的时候几M吧，但慢的时候也挺慢的。有时会提示下载错误，多刷新几次就行了。

这个方法下载时有时半路会提示失败，这时就又需要暂停一下，刷新一下才行，心累……

然后我又找到了一个办法，直接在GitHub上download zip可以有1M多，再也不是几百b和几k了！

方法就是把git的代理删除掉。
打开cmd，输入
git config –global –unset http.proxy
git config –global –unset https.proxy
这样就可以了，速度快得飞起！
