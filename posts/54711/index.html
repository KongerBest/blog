今天介绍一款仿 macOS UI 的软件：

——MyFinder、MyDock。

适用于装逼或是真的喜欢 macOS UI，

但因为某些原因不得不用 Windows 的用户。

先来看看软件官网效果图

再看看笔者实测的整体效果

苹果公司于2019年 6 月发布了最新的Macintosh桌面操作系统，并于 2019 年 10 月 8 日正式推出macOS Catalina。

苹果 Newsroom 展示：

结束了 iTunes 时代的 macOS Catalina 实际体验如何？

macOS Catalina （版本 10.15）是苹果公司用于麦金塔计算机的桌面操作系统 macOS 的第 16 个 主要版本。它是 macOS Mojave 的继任者。Catalina 是第一个只支持 64 位应用程序的 macOS 版本。

https://zh.wikipedia.org/wiki/MacOS_Catalina

为什么有那么多人喜欢苹果电脑呢?

我觉得 macOS 的界面友好易用是一方面

我们看一下维基百科的描述

尽管最重要的架构改变是在表面之下，但是 Aqua GUI 是最突出和引人注目的特色。柔软边缘的使用，半透明颜色和细条纹（与第一台 iMac 的硬件相似）把更多的颜色和材质带入到桌面上的视窗和控件，比 OS9 所提供的外观更多。

https://zh.wikipedia.org/wiki/MacOS

好啦，下面开始详细介绍软件
软件官网标语——体验极致模仿 mac os 系统桌面

点击查看大图，是不是基本上看不出区别？

https://www.mydockfinder.com/www.mydockfinder.com

够简洁又够复杂，为什么这样说呢？
下面分开来介绍

一、MyFinder

左端会显示当前窗口程序名

鼠标移到左上角点击显示开始菜单

有些栏目是为了模仿而设置的

（无法点击使用，即使显示黑非灰）

下面两个栏目是有较多的可用项目

比如：偏好设置

开启 MyFinder 后，理论上可以隐藏任务栏了
因为 MyFinder 有任务栏大部分的功能

1、CPU、磁盘、内存状态显示

2、网络 IP 和无线网络连接传输等情况

3、音量调整显示

4、系统托盘与时钟

5、输入法和虚拟键盘

6、快速联网搜索

MyFinder 是不是够简洁又够复杂？

二、MyDock

我们看看动态效果

窗口最小化和打开时显示过渡动画

可以向 Dock 添加图标

如果觉得图标不好看，可以自己找 icon 文件修改

添加天气、时间、控制面板、托盘图标等工具

以添加天气为例

Dock 会显示正在运行的软件图标

如果嫌图标太多，可以设置不显示

Dock 偏好设置

第一次使用可能会偶尔崩溃，

建议安装后重启使用

如果使用了几天仍然频繁崩溃
那可以使用另一款替代品
—— 软媒桌面（mydesk）
只是功能比较简单

感觉字数不够，水了点壁纸（滑稽）
